Projects

Rock Paper Scissors Game (Java)

This project is a simple console-based Rock Paper Scissors game developed using Java. The game allows the user to choose between rock, paper, or scissors, while the computer randomly generates its choice.

The program compares both choices, determines the winner based on game rules, and displays the result. This project helped me understand basic Java concepts such as conditional statements, loops, user input, and random number generation.
